# findata-avan

A wrapper for the Alpha Vantage API with the goal of mapping the JSON to crystal objects.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     findata-avan:
       github: sumwatt/findata-avan
   ```

2. Run `shards install`
